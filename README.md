# Install

1. Clone the repository
2. Execute
```
npm install
```
3. You need to have Nest CLI installed
```
npm install -g @nestjs/cli
```
4. Run DB throught docker compose
```
docker-compose up -d
```


## Stack
* MongoDB
* NestJS
* Typescript
